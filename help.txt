This program can sort strings of famous "Eugene Onegin" by A. Pushkin.

1. It sorts all strings in lexicographic order from the beginning skipping all punctiation marks and spaces.
For example: "AB><>! and AC is sorted in lexicographic order
2. Then, it sorts all strings in lexicographic order from the end skipping all punctuation marks and spaces.
For example: "<>AB<"""AAA and "<>!@<>$$>@<!<!AAB is sorted in lexicographic order from the end.
3. Finally, It writes in output.txt original poem.
P.S. Each part of output.txt is diveded by "--------------------------------"
